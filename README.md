# Setup for local development

clone the repo
```
git clone git@github.com:yk-jp/expense-tracker.git
```

move to the root dir
```
cd expense-tracker
```

run a docker file

```
make run
```

You can see collections in mongodb database at 
```
localhost:27018
```

open 
 ```
 http://localhost:3000
 ```
